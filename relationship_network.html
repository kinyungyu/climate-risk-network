One entity per line: id | label | type | category | pages | detail
type controls the color. Define types in Settings.
category is used for filtering (e.g. segment, department).
pages is the source document page reference (e.g. "5, 9-10").
detail is shown on hover (supports <strong> tags).
One relationship per line: from_id | to_id | label
Use the entity id (first column) from the Entities tab.
Define entity types and their colors. One per line: type_key | Display Label | #hexcolor
The first type is used as the "center" node (drawn largest).
Example: center | My Company | #4a9eff
